## Recommendations
1. **Parameterise** all login/database queries; never concatenate user input into SQL.
2. Remove the `password`/`totpSecret`/`deluxeToken` columns from every user-facing API response. Enforce admin-only RBAC on `/api/Users` and `/rest/user/authentication-details`.
3. Apply object-level authorisation checks on `/rest/basket/{id}`, `/api/Cards`, `/api/Addresss`, `/api/Users/{id}` (compare `req.user.id` to resource owner).
4. HTML-encode the search term; stop rendering `q` via `innerHTML`.
5. Disable `serve-index` on `/ftp`; move backups off the web root; reject filenames containing null bytes and double-encoded sequences.
6. Require authentication on `/rest/admin/*` endpoints.
7. Upgrade/replace `express-jwt@0.1.3`, `sanitize-html@1.4.2`, and other pinned vulnerable dependencies.
